# Big Data Visualization Project

## Overview
This project focuses on visualizing Teen Mental Health data using Python libraries. Different types of graphs and charts are used to analyze social media usage, sleep patterns, academic performance, and physical activity among teenagers.

## Objectives
- Analyze teen mental health dataset
- Understand social media usage trends
- Visualize academic performance and sleep habits
- Create meaningful graphical insights from data

## Technologies Used
- Python
- Pandas
- Matplotlib
- Google Colab
- GitHub

## Dataset Used
Teen Mental Health Dataset

## Visualizations Included
- Bar Graph
- Pie Chart
- Histogram
- Scatter Plot
- Line Graph
- Box Plot
- Correlation Heatmap

## Project Workflow
1. Dataset Collection
2. Data Loading using Pandas
3. Data Cleaning and Processing
4. Data Visualization using Matplotlib
5. Uploading Project to GitHub

## Conclusion
This project successfully visualizes important patterns related to teen mental health and social media behavior using multiple graphical techniques.
